# libLCE

A TS library for Minecraft Legacy Console Edition files + more 

![Progress](/assets/cur_progress.png)

## Progress

- [ ] Console compression types - 50% complete
- [x] Savegame parsing
- [x] Savegame creation
- [ ] World info parsing - 50% complete
- [ ] World folder parsing
- [ ] Region parsing
- [ ] Region creation
- [ ] Chunk parsing
- [ ] Chunk creation
- [ ] Other file formats
